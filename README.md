## Overview
This is a personal portfolio website showcasing skills, projects, experience, and contact information. It is fully responsive and built with modern web design practices. The design emphasizes readability, aesthetics, and interactivity with smooth transitions and hover effects.

The website includes sections such as:
- Home
- About
- Services
- Experience
- Portfolio / Projects
- Skills
- Contact
- Footer

## Features
- **Responsive Design:** Adapts seamlessly to desktops, tablets, and mobile devices.
- **Smooth Navigation:** Smooth scroll and sticky header for easy access to sections.
- **Animated Elements:** Hover animations on buttons, project cards, services, and skills.
- **Contact Form:** Fully styled form for users to get in touch.
- **Portfolio Showcase:** Grid-based project display with images, descriptions, and links.
- **Experience Timeline:** Visually appealing timeline for experience and milestones.
- **Statistics & Tagline:** Eye-catching stats section with professional tagline.
- **Footer & Social Links:** Includes contact info, quick links, and social icons.

## Technologies Used
- **HTML5** – Semantic structure
- **CSS3** – Modern styling, grid & flex layouts
- **Google Fonts** – 'Montserrat' and 'Bebas Neue' for typography
- **Font Icons** – Boxicons (for menu and service icons)
- **Responsive Design** – Mobile-first media queries
